WebConvers

What is webconvers? :

webconvers is a web-based interface to the amprnet World Wide Convers system that only requires a browser capable of supporting frames. With the addition of Javascript (selectable by the installer or by the user at login time) a lot of additional facilities become available that improve performance through caching proxy servers, improved display and more functionality.
NOTE: this is not a telnet or AX25 gateway, it provides an interface into WW convers - thats all.

webconvers is written in the PHP language, which is fairly cross-platform, and available for many different web servers. The code is released under version 2 of the GPL.

Features:

  • A lot more than I originally intended! All I wanted was to get into convers from work through the company firewall!!
  • Customizable, multi-lingual strings.
  • Enhanced interface for JavaScript enabled browsers.
  • User list of who is on the channel
  • Single click private messages to other users
  • Access control by host, domain, network and callsign.
  • Channel listing window that allows selection of channel, leaving a channel and private messaging

Flaws:

  • Scalability Unknown.
  • Doesn't work with IE5 for Macintosh, among others.

FAQ:

Q. Is webconvers an IRC client?
A. No. Try phpChat.

Q. How do I install webconvers?
A. Extract the archive in to some directory and look at config.php. You'll want to edit the configuration variables.
